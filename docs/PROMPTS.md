# Prompts for Reusable Studio Engine

## User Prompts
1. **What is your main goal with this project?**
   - Describe the primary objective you want to achieve.

2. **What features do you want to implement?**
   - List the key functionalities you envision for the application.

3. **Who is the target audience for this project?**
   - Define the users who will benefit from this application.

4. **What platforms do you intend to support?**
   - Specify whether the application will be web-based, mobile, or desktop.

5. **What are the expected user interactions?**
   - Outline how users will interact with the application (e.g., mouse clicks, keyboard inputs).

## Developer Guidelines
1. **Code Structure**
   - Maintain a modular code structure for easy maintenance and scalability.

2. **Documentation**
   - Ensure that all functions and modules are well-documented for clarity.

3. **Version Control**
   - Use version control practices to track changes and collaborate effectively.

4. **Testing**
   - Implement unit tests for critical components to ensure reliability.

5. **Performance Optimization**
   - Focus on optimizing performance, especially in rendering and user interactions.
